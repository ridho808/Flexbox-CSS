FLEXBOX

definisi flexbox

flexbox adalah membuat layout 1 dimensi yang dapat
mengatur jarak dan penjajaran antara item dan sebuah container


istilah dalam flexbox

  • Main axis: sumbu utama dari sebuah container yg menentukan urutan dari penempatan item secara horizontal
  • Main start | Main end : mulai dan berakhirnya items yg disimpan dicontainer
  • Main size: ukuran (width/height) dari container yang akan membuat dimensi dari items-nya relatif terhadap main size
